Database migrations for .NET. Based on the idea of Rails ActiveRecord Migrations.

## Supported Databases

* MySQL
* Oracle (not well tested?)
* PostgreSQL
* SQLite
* SQL Server

## Supported Modes

* [[NAntTask]]
* [[MSBuildTarget]]
* Console Application (You should be using an automation tool! :))
